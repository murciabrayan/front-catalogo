# Frontend - Catálogo Makyp

Frontend del catálogo de `makyp_creations`.

Aquí está la parte visual del proyecto: landing, catálogo público, modales de productos, generación de PDF y panel administrativo conectado con la API.

## Tecnologías

- React
- TypeScript
- Vite
- CSS

## Qué incluye

- portada con enlaces a redes
- catálogo público por categorías
- detalle de productos en modal
- cálculo de adicionales y total
- descarga del pedido en PDF
- panel administrativo con login

## Ejecutarlo en local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Nota

Este frontend consume la API del backend. Para probar toda la experiencia completa, el backend también debe estar corriendo.
